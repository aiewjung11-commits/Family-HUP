📲 ติดตั้งเป็นแอปบนมือถือ

Android (Chrome): เปิดลิงก์ https:// ของหน้านี้ → กดปุ่ม "ติดตั้งแอปเลย" หรือเมนู ⋮ → เพิ่มลงในหน้าจอหลัก / ติดตั้งแอป
iPhone (Safari): เปิดลิงก์ → กดปุ่มแชร์ → เพิ่มลงในหน้าจอโฮม (Add to Home Screen)
✅ ได้ไอคอน 🏠 บนหน้าจอ เปิดเต็มจอเหมือนแอปจริง เปิดใช้ offline ได้หลังเข้าครั้งแรก
⚠️ ต้องเปิดจากลิงก์ https:// (เช่น GitHub Pages) — เปิดจากไฟล์ในเครื่องจะติดตั้งไม่ได้
💡 วิธีใช้ให้ทุกคนในบ้านเข้าได้: อัปโหลดไฟล์นี้ขึ้น GitHub Pages (แบบเดียวกับ MoneyPad) → คัดลอกลิงก์มาวาง → สร้าง QR → พิมพ์แปะตู้เย็น 🧲 ทุกคนสแกนเปิดจากมือถือได้เลย
⚠️ ข้อมูลเก็บใน localStorage ของแต่ละเครื่อง — ถ้าต้องการข้อมูลชุดเดียวกันทุกเครื่อง ใช้ปุ่ม Backup/Restore หรือรอเชื่อม Firebase
